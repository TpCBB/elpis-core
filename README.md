# Eplis

一个基于 Vue 3 + Koa2 的全栈项目。

## 技术栈

- 前端: Vue 3 + Vue Router + Pinia + Element Plus
- 后端: Koa2
- 数据库: MySQL
- 构建工具: Webpack 5

## 开始使用

### 安装依赖

```bash
npm install
```

### 启动BFF服务

```bash
npm run dev
```

### 启动前端服务

```bash
npm run build:dev
```

### 环境要求

- Node.js (建议 v14 或更高版本)
- MySQL
